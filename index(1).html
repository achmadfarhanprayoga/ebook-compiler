HTML

Pengertian HTML

HTML (Hyper Text Markup Language) adalah sebuah bahasa formatting yang digunakan untuk membuat sebuah halaman website. Di dalam dunia pemrograman berbasis website(Web Programming), HTML menjadi pondasi dasar pada halaman website. sebuah file HTML di di simpan dengan ekstensi .html (dot html) atau sering ditandai dengan nama index.html. dan dapat di eksekusi atau diakses menggunakan web browser(Google Chrome, Mozilla Firefox, Opera, Safari dan lain-lain).

HTML merupakan singkatan dari HyperText Markup Language. Dengan HTML, Anda dapat membuat halaman web yang terdiri dari teks, gambar, tautan, formulir, dan banyak lagi. Berikut adalah beberapa elemen dasar dalam HTML:

  • <html>: Menandakan awal dan akhir dari dokumen HTML.
  • <head>: Berisi informasi tentang dokumen, seperti judul dan meta tag.
  • <body>: Berisi isi dari dokumen, seperti teks, gambar, dan elemen-elemen lainnya yang akan ditampilkan di halaman web.
  • <h1>, <h2>, ..., <h6>: Menandai heading atau judul.
  • <p>: Menandai paragraf teks.
  • <a>: Membuat tautan atau hyperlink.
  • <img>: Menyisipkan gambar.
  • <ul>: Membuat daftar tak-terurut.
  • <li>: Membuat item dalam daftar.
  • <form>: Membuat formulir untuk interaksi pengguna.
  • </..>: untuk menutup perintah, contoh </p>

    CSS

    Pengertian CSS

    Setelah mempelajari HTML kita akan mempelajari CSS untuk memberikan styling atau mempercantik tampilan pada html CSS digunakan untuk mengatur tampilan dan layout dari elemen-elemen HTML. Dengan CSS, Anda dapat mengatur warna, ukuran, posisi, dan efek visual lainnya pada halaman web. CSS juga memungkinkan Anda untuk membuat desain yang responsif, yang dapat menyesuaikan tampilan halaman web dengan berbagai perangkat dan ukuran layar...

    CSS terbagi kedalam 3 jenis yaitu

  • Inline CSS Merupakan kode yang ditulis didalam file html dan menggunakan elemen spesifik yang memuat tag <style>. Jenis CSS ini hanya mempengaruhi satu baris kode HTML dan harus menempel pada elemen tulisan tersebut.
  • Internal CSS Merupakan kode yang dituliskan pada bagian header file HTML. Sayangnya jenis ini membuat loading semakin lama.
  • External CSS Merupakan kode CSS yang diletakkan diluar dokumen CSS sebgai file .css. External CSS memiliki fungsi untuk mengkustomisasi semua tampilan halaman website yang sudah ditentukan atau beberapa halam sekaligus.

    • CSS adalah kepanjangan dari Casscarding Style Sheet, CSS bisa di aplikasikan langsung didalam file HTMl bisa pula di aplikasikan terpisah dengan nama style.css. Berikut adalah elemen elemen dasar dalam penulisan CSS:

    Elemen penggunaan CSS:

    Biasanya dimasukan dalam tag <head >
    Untuk merubah tampilan di bagian tertentu dengan cara menulis bagian yang ingin di perbagus seperti contohnya
    body{

    Bab 3: JavaScript

    JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web menjadi interaktif. Dengan JavaScript, Anda dapat menangani interaksi pengguna seperti klik tombol, mengisi formulir, dan berinteraksi dengan elemen-elemen HTML pada halaman web.

    JavaScript juga memungkinkan Anda untuk membuat animasi, validasi formulir, memuat konten secara dinamis, dan banyak lagi. Hal ini membuat pengalaman pengguna menjadi lebih dinamis dan menarik.

    Contoh penggunaan JavaScript:

                        
                        // Menampilkan pesan saat tombol di klik
                        document.getElementById("myButton").addEventListener("click", function() {
                            alert("Tombol telah diklik!");
                        });

    Dalam contoh di atas, JavaScript digunakan untuk menambahkan event listener ke sebuah tombol. Ketika tombol tersebut diklik, akan muncul pesan peringatan.
